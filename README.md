 Demo App
### Overview 
A multi tiered application with javascript & Postgres. Get up and running asap with local containers to k8s. 
*** MUST HAVE DOCKER INSTALLED ***

### Install
Step 1:
```
 git clone https://github.com/thecloudslayer/demoapp.git
 ```
### Navigate 
Step 2: 
```
 cd demoapp
```

### Run
Step 3:
```
 docker-compose up
```

## Usage
```
http://localhost:3000/
```
